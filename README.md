Impurities Detection in Edible Bird’s Nest

This project uses MATLAB to detect impurities in edible bird’s nest images through optical segmentation and image fusion techniques.

Cite this article

Yee, C.K., Yeo, Y.H., Cheng, L.H. et al. Impurities detection in edible bird’s nest using optical segmentation and image fusion. Machine Vision and Applications 31, 68 (2020). https://doi.org/10.1007/s00138-020-01124-y

Usage

Run the MATLAB scripts in the code/ folder to process images and generate results.
